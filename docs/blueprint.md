# **App Name**: Formula Skills Garage

## Core Features:

- Student Free Access: Students can freely access TP content, racing simulations, and skill-building exercises.
- Teacher Secured Access: Teachers access a secured area with password 'Mongy' for student progress tracking, competence evaluation, settings, and data management.
- Competence Tracking & Evaluation: Teachers can track individual student progress and evaluate competence based on TP completion and racing simulation performance.
- Configurable Parameters: Teachers can adjust parameters for TPs, simulations, and scoring to tailor the learning experience.
- Data Recording & Reset: Teachers can record student data, save progress, and reset data for new semesters or evaluations.
- AI-Powered Skill Gap Analysis: The tool analyzes student performance data to identify skill gaps and suggests personalized learning paths.
- Adaptive Racing Simulations: Racing simulations adapt in difficulty based on student performance, providing a challenging but achievable learning experience.
- Collaborative TP Design: Teachers can create and share customized TP modules with integrated racing simulations.

## Style Guidelines:

- Primary color: Racing red (#FF0000) to evoke the speed and energy of Formula 1.
- Secondary color: Carbon fiber black (#333333) for a sleek, modern aesthetic.
- Accent color: Electric blue (#7DF9FF) to highlight interactive elements and provide contrast.
- Headline font: 'Bebas Neue' for a bold, dynamic feel.
- Body font: 'Roboto' for clear, readable text.
- Use motorsport-inspired icons for skills, tools, and car components.
- Dashboard layout inspired by a Formula 1 cockpit, with key information easily accessible.
- Use fast, dynamic animations to simulate speed and excitement.
